# Unity Learn: Prototype 1

## Overview
This repository contains my implementation of Prototype 1 from the official Unity Learn pathways.

## The Game
* Involves a moving vehcile on a flat plane.
* There are crates as obstacles, which react to collisions.
* Move with arrow keys or WASD.
* As an addition, I have implemented a speed up function. When shift key is pressed, the vehicle will move faster.

## What I Learned
* Navigating the Unity Editor interface.
* Applying basic physics and rigidbodies to GameObjects.
* Capturing and scripting basic player input.
